## 🚀 **Quick Start**

### **1. Install Dependencies**
```bash
pip install -r requirements.txt
```

### **2. Configure Bot**
Edit `core/config.py` and add your Discord bot token:
```python
DISCORD_TOKEN = "your_discord_bot_token_here"
```

### **3. Start the Bot**
```bash
python start.py
```
